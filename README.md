# Python Design Patterns

A collection of Python implementations for classic design patterns.

## Requirements

- Python 3.8+

## How to run

Run any pattern example directly:

```bash
python3 Singleton.py
python3 FactoryMethod.py
python3 Observer.py
```

To run all examples in sequence:

```bash
for f in *.py; do
  python3 "$f"
done
```
